Tic Tac Toe

A simple browser-based implementation of the classic **Tic Tac Toe game** built using web technologies. The project demonstrates fundamental game logic, DOM manipulation, and interactive UI behavior.

🎮 Features

* Two-player gameplay
* Interactive game board
* Win detection logic
* Draw detection
* Game reset functionality

🛠 Tech Stack

* HTML
* CSS
* JavaScript

🚀 Getting Started

1) Clone the repository
```bash
git clone https://github.com/gadilkartanishka/tic-tac-toe.git
cd tic-tac-toe
```

2) Run the project

Simply open the `index.html` file in your browser.

🎯 Learning Objectives
This project was built to practice:
- JavaScript DOM manipulation
- Event handling
- Game logic implementation
- Frontend UI interaction
